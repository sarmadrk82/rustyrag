# RustyRAG Getting Started

RustyRAG is a config-first RAG pipeline framework written in Rust.

## What it does

1. Reads documents from a folder
2. Splits them into chunks
3. Embeds each chunk with OpenAI
4. Stores vectors in Qdrant

## Run the pipeline

1. Copy the example env file and add your OpenAI key:

```bash
cp .env.example .env
# edit .env and set OPENAI_API_KEY
```

2. Start Qdrant and run the pipeline:

```bash
docker compose up -d
cargo run -p rustyrag-cli -- etl run configs/pipelines/docs-index.yaml
```

The CLI loads variables from `.env` automatically when you run commands from the project root.

After indexing, Phase 2 will add a query API to ask questions against your documents.
